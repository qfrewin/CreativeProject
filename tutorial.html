Tutorial

I figure for another one of my pages to meet the grading requirements for the first submission, that I will give a better idea of what my program looks like and how to use it.

Plan Page

This page is only ever seen by the person making the schedule. One can put the different locations, tables, etc. at the top of the page. The time increments go on the left. The user can then easily type in the name of a class in any corresponing location/time box in the spreadsheet. Some locations may be a combination of two or more others, so if a class is entered into one of these fields, it will black out any potentially interfering locations inside of that time slot.

Class Page

This page is only used by the scheduler. The names of the various classes are written at the top of the page with the time increments going down the left side of the page. The user can type the name of a coach under a class to signify that that coach teaches that class at whatever time that row corresponds to. The program ensures that a coach is never doubly enterred into a time slot, though that is something I want to change to be an optional setting someday.

Coach Page

This is by far the most useful page. This page compiles the information from the other two and presents it in a more meaningful way for the employees. A coach, waiter, etc. can look at the sheet and easily tell where they need to be, when to be there, and what they are doing.
